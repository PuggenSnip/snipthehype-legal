Snip The Hype

A personal, single-user tool that edits my own Twitch gaming highlight clips into short vertical videos and publishes them automatically to my own channels:

  • YouTube Shorts — @SnipTheHype
  • Instagram Reels — @snip.the.hype
  • TikTok — via the official Content Posting API

Every clip credits the original streamer and clipper on screen and in the description, with a link back to the source clip.
